This is my blog project. I will be creating a blog website that will be running on Django web framework with python. 
The requrements for first release are:
1. Blog should be able to display media content: 
    *Video
    *Audio
    *Photos
2. Reader interaction with Blog post:
    *User comments
    *Allow end user to edit comments
3. Create search function for historical blogs 
4. Integrate rich text editor with Django to allow content editing
    *Integrate functionality for code snippets on blog post
5 Create a "about" page for author
6 Create index page for blog posts
